# IncubationPeriodModels

## IncubationPeriodModels 0.1.1

* Initial release of the package.

* Implements Burr Type III, Burr Type X, Burr Type XII and derive Burr distributions.

* Provides density (`d*`), distribution (`p*`), quantile (`q*`), and random
  generation (`r*`) functions for Burr-family distributions.

* Includes a derived Burr-like distribution for modelling incubation periods
  and other epidemiological delay distributions.

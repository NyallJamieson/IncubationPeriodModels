# Burr distributions (Type III, X, XII) and a derived Burr-like distribution

This package implements density, distribution, quantile, and random
generation functions for several Burr distributions and a derived
Burr-like distribution.

## Details

Included families:

- Burr Type III: `dburr3`, `pburr3`, `qburr3`, `rburr3`

- Burr Type X: `dburr10`, `pburr10`, `qburr10`, `rburr10`

- Burr Type XII: `dburr12`, `pburr12`, `qburr12`, `rburr12`

- Derived distribution: `dburrNEW`, `pburrNEW`, `qburrNEW`, `rburrNEW`

All distributions are supported on \\x \> 0\\.
